## Workshop data

`1971.csv.gz` is the prepared 1971 NCHS natality birthweight file used by the
Unit 3 birthweight notebook. It was generated from the CDC natality archive
referenced in `DAIR3-Workshop/resources/unit_3/prep.py`, using the same column
specification and recodes for the workshop-relevant columns.

The Unit 4 notebooks load this file through `materials-draft/setup.py`.
